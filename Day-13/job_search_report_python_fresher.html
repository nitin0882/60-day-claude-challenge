Skill gap analysis (vs. your current profile)
SQL — critical gap
Required in 75% of listings. Your roadmap already identifies this as Week 1–2. Prioritise SQLZoo / Mode Analytics immediately — this single skill unlocks the majority of roles.
Git & GitHub — critical gap
Every employer checks this. No public GitHub = invisible to technical recruiters. Your roadmap places this in Week 9–10 — consider moving it to Week 1 alongside SQL.
Docker — partial gap
Appears in ~52% of JDs, especially startups. Basic containerisation knowledge (build, run, push) is enough for fresher roles. A free Docker 101 course addresses this in ~4 hours.
Cloud basics (AWS / GCP) — partial gap
Nice-to-have at fresher level, not blocking. AWS Free Tier + deploying one Flask app to EC2 or Render.com is sufficient to mention on your CV and LinkedIn.
Recommendations to improve interview chances
1Move Git to Week 1 — create your GitHub today, push even basic scripts. Recruiters screen GitHub before interviews. A blank profile loses you roles before they even see your resume.
2Build and deploy one visible project end-to-end (e.g. a Flask REST API deployed on Render or Railway). A live URL on your resume beats 10 offline projects.
3Apply on Internshala and Cutshort — both are strong for Delhi NCR startups hiring Python freshers. Set job alerts for "Python developer", "SDE-I Python", and "Django fresher".
4Optimise LinkedIn headline to "Python Developer | Django | Flask | REST APIs | Open to Work" and turn on Open to Work. Recruiters from Noida/Gurgaon actively search these keywords.
5Target ₹3.5–5 LPA as your realistic first-offer range (CTC). Your ₹6 LPA ceiling is achievable after 6–12 months of experience, or immediately at a product startup if you have strong GitHub projects.
